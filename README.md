Description
This project is a C++ based system that combines concepts of discrete mathematics and data structures in one program. It includes implementation of sets, logic operations, graph algorithms, relations, and functions. The main goal of this project is to understand how theoretical concepts like set theory, graph theory, and logic can be applied in programming.
Features
Custom implementation of sets using dynamic arrays (without using STL set)
Set operations like union, intersection, difference, power set, and Cartesian product
Logic engine supporting For All, Exists, and logical law verification
Graph implementation using adjacency matrix and list
Algorithms like Dijkstra (shortest path) and Kruskal (minimum spanning tree)
Detection of Euler path, Euler circuit, and Hamiltonian cycle
Relations and functions checking (reflexive, symmetric, injective, bijective)
Menu-driven interface for user interaction
Technologies Used
C++
Object Oriented Programming
Data Structures and Algorithms
Purpose
The purpose of this project is to strengthen understanding of DSA and discrete structures by implementing all concepts manually and integrating them into a single system.
